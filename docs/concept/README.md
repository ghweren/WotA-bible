## Обзор
Этот раздел определяет фундаментальные принципы проекта "Колесо Апокалипсиса" - от краткого описания до целевой аудитории.

## Содержание
- [1.1. Лифт-питч](elevator-pitch.md) - суть проекта в 3-5 предложениях

## Ключевые принципы
1. **Гибридность** - соединение глобальной стратегии и тактических сражений
2. **Нарративная глубина** - история как двигатель игрового процесса
3. **Моральная неоднозначность** - отсутствие "правильного" выбора
4. **Технологическая реализация** - баланс между масштабом и управляемостью
